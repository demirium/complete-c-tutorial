# 📚 CMPE113: Introduction to C Programming

A comprehensive, beginner-friendly guide to C programming fundamentals, created with real student experiences in mind.

---

## About This Project

This educational resource was developed based on my experience with university-level C programming courses (CMPE113 & CMPE114). It addresses the common challenges and learning gaps I encountered during my studies.

**Why I created this:**
- To help underclassmen navigate the same difficulties I faced
- 🎓 To provide a clear, structured learning path for C beginners
- 🌍 To offer a free, accessible resource for anyone learning C
- 📂 To build a meaningful portfolio project
- 🤝 To give back to the programming education community

This is **CMPE113**, the foundational course covering C programming basics. CMPE114 (advanced topics) is coming soon!

---

## 📖 What's Inside?

This course covers the fundamental concepts of C programming in a logical, easy-to-follow sequence. Each section includes:

- 📝 Detailed explanations and concept breakdowns
- 💻 Multiple working code examples
- Common beginner mistakes and how to avoid them
- Practice suggestions and exercises
- ❓ FAQ sections addressing typical confusions

---

## 🗂️ Course Contents

### **Fundamentals**
1. **[Introduction](1%20-%20Introduction)** - Your first C program, basic syntax, and program structure
2. **[Variables](2%20-%20Variables)** - Data types, declarations, and placeholders
3. **[Getting Inputs](3%20-%20Getting%20Inputs)** - User interaction with `scanf`

### **Control Flow**
4. **[If / Else Statements](4%20-%20If%20Else)** - Conditional logic and decision making
5. **[Switch Case](5%20-%20Switch%20Case)** - Multi-way branching and menu-driven programs

### **Loops**
6. **[For Loop](6%20-%20For%20Loop)** - Counted repetition and iteration patterns
7. **[While Loop](7%20-%20While%20Loop)** - Condition-based repetition and input validation

### **Standard Libraries**
8. **[Math Library](8%20-%20Math%20Library)** - Mathematical functions with `math.h`
9. **[Stdlib Basics](9%20-%20Stdlib%20Basics)** - Random numbers and utilities with `stdlib.h`
10. **[Functions](10%20-%20Functions)** - Reusable code blocks with `functions`

## How to Use This Resource

1. **Start from the beginning** - The sections are designed to build on each other
2. **Read the README** - Each section has a comprehensive README.md explaining the concepts
3. **Study the examples** - Run and modify the example code files (ex-1.c, ex-2.c, etc.)
4. **Practice** - Try the suggested exercises and build your own variations
5. **Experiment** - Break things! Understanding errors is part of learning

---

## 🛠️ Getting Started

### Prerequisites
- A C compiler (GCC, Clang, or MSVC)
- A text editor or IDE (VS Code, Code::Blocks, Dev-C++, etc.)
- Basic computer literacy
- **No prior programming experience required!**

(I recommend using VSCode with the C/C++ extension. If you are a MacOS user, you can use Xcode. This is the best way to get started with C programming.)
