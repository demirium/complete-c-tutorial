# Project Samples

Welcome to the **Project Samples** directory! 

Here, you'll find various projects and code snippets that pull together everything you've learned in CMPE113 and CMPE114. 
Think of these as the "mid-season finale" episodes of your coding journey—where all the character arcs (variables, loops, pointers) finally come together.

## 📁 What's inside?

- `basic-gpa-calculator.c`: A practical, real-world example of using structs, arrays, logic, and loops to calculate a student's GPA.

## 🧠 Why look at these?

Learning syntax is one thing, but putting it all together in a larger program is where the real learning happens. 
These samples are designed to show you how "textbook" concepts look like when solving actual problems.

*Noice!* Now go ahead and read the code, modify it, break it, and fix it again. That's the best way to learn!
