# Lab Exam Questions 📝

Welcome to the **Lab Exam Questions** directory! 

This folder is a goldmine. It contains past lab exams and practical coding questions from my school's C programming courses. 

## 🧠 Why is this here?
Exams can be stressful, but practicing with actual past questions is the ultimate cheat code (a legal one, I promise). I'm sharing these so you can feel the pressure of a real lab environment and test if you *truly* understand the concepts.

I'll be updating this folder with more exams as I dig them up from my archives.
